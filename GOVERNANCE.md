# Governance

`get.rpath.dev` is maintained by the rpath maintainers.

Maintainers are responsible for:

- Reviewing installer and uninstaller changes.
- Keeping deployment configuration safe and minimal.
- Responding to security reports.
- Coordinating releases with the main `rpath` repository.

Major changes to install location, script trust model, checksum behavior, or default shell-wrapper behavior should be discussed in an issue before implementation.
